# 📊 Sales Performance Dashboard

## 🚀 Project Overview

This project focuses on analyzing retail sales data to extract meaningful business insights and building an interactive dashboard using Power BI.

The workflow includes data cleaning, exploratory data analysis (EDA), and visualization to support data-driven decision-making.

---

## 🎯 Objectives

* Analyze overall sales performance
* Identify top-performing products and regions
* Track monthly sales trends
* Compare sales and profit to evaluate business efficiency
* Provide actionable business insights

---

## 🛠️ Tools & Technologies

* **Python** (Pandas, NumPy, Matplotlib)
* **Power BI** (Dashboard & Data Visualization)
* **Excel / CSV Dataset**
* **Git & GitHub** (Version Control)
